*Mobile Charter*

Healthcare Charter project for Nurses and Hospitals in Canada <br/><br/>
TO GET STARTED
--------------
Step 1:<br/>
Go to Branch experiment/NextJS on github, then create a branch in your name in this format: "{Name}sBranch" e.g tobisBranch. Then clone on your system.

Step 2:<br/>
Ask me for the database .env variable, create and paste it in a file called .env

Step 3:<br/>
Run:<br/>
npx prisma generate
then npx run dev

Step 4:<br/>
View "Issues" tab on repository to see assigned tasks.

And Lastly:<br/>
ASK QUESTIONS WHEN STUCK!!; can't stress the value of this enough, it helps us grow as developers and also makes us complete tasks as fast as possible.
